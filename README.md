## Getting Started

1. Clone the repo
\`\`\`bash
git clone https://github.com/yourusername/NebulaNet.git
\`\`\`

2. Install dependencies
\`\`\`bash
npm install
\`\`\`

3. Set up environment variables
\`\`\`bash
cp .env.example .env
\`\`\`

4. Start the development server
\`\`\`bash
npx expo start
\`\`\`
